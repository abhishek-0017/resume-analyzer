# AI Resume Analyzer

A full-stack web application that analyzes resumes and provides ATS score, skill matching, and improvement suggestions.

---

## Features
- User Authentication (Signup/Login)
- Upload Resume (PDF)
- ATS Score Calculation
- Skill Matching & Missing Skills Detection
- AI-based Feedback Suggestions
- User History Dashboard

---

## 🛠 Tech Stack
- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MongoDB
- File Handling: Multer, PDF-Parse

---

## 📊 How It Works
1. User uploads resume
2. Backend extracts text from PDF
3. Skills are matched with predefined list
4. ATS score is calculated
5. Results are saved and displayed
