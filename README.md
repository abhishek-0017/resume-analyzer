# AI Resume Analyzer 

A full-stack web application that analyzes resumes using ATS (Applicant Tracking System) logic and matches them against job descriptions.

## Features
- Upload Resume (PDF)
- Paste Resume Text
- ATS Score Calculation
- Job Description Matching
- Missing Keyword Detection
- Real-time Progress Bars

## Tech Stack
- Frontend: React.js
- Backend: Node.js, Express.js
- PDF Parsing: pdf-parse
- Deployment: Vercel (Frontend), Render (Backend)

## Live Demo
Frontend: https://resume-analyzer-8n1a.vercel.app  
Backend: https://resume-analyzer-backend-9dde.onrender.com
